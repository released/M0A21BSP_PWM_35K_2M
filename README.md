# M0A21BSP_PWM_35K_2M
 M0A21BSP_PWM_35K_2M

update @ 2022/06/23

1. Initial PA2(PWM0_CH1) , PC0(PWM0_CH4) on M0A23 EVB ,

2. PA2(PWM0_CH1) , freq : 31.25K , use PWM discrete function to change duty

3. PC0(PWM0_CH4) , freq : 2M , use PWM driver to change duty

4. use terminal , letter a/A/d/D , to change PA2(PWM0_CH1) duty

5. use terminal , digit 1/2 , to change PC0(PWM0_CH4) duty
